## 🔧 Setup Steps

- [ ] 1. Install Node.js from nodejs.org (if not installed)
- [ ] 2. Verify Node.js: `node --version` and `npm --version`
- [ ] 3. Save three images to `/public/images/` folder
- [ ] 4. Open Terminal and navigate to project folder
- [ ] 5. Run `npm install`
- [ ] 6. Run `npm run dev`
- [ ] 7. Site opens at http://localhost:3000
